# Changelog

## 0.1.0 - 2025-08-30

Initial functional release

### Added

1. Top bar `Workspaces` indicator with popup menu
2. Recent VSCode workspace discovery from local storage
3. Workspace menu actions: `open`, `view all`, `settings`
4. Preferences UI with projects folder path, limit, and remove controls
5. Git and folder icon support with symbolic assets

### Behavior

1. Workspace list refreshes every 60 seconds
2. Removed workspaces are hidden from menu and preferences list
3. Reopened workspaces return to the visible list
