# Changelog

## 0.1.0 - 2025-09-03

Initial functional release

### Added

1. Top bar `GitHub` indicator with popup menu
2. Recent GitHub repositories discovery via REST API
3. Repository menu actions: `open`, `view profile`, `settings`
4. Preferences UI with PAT entry and limit controls
5. Ownership, collaborator, and fork icon support with symbolic assets

### Behavior

1. Repository list refreshes every 60 seconds
2. Invalid PAT shows error message in menu
3. No PAT shows token creation button linking to GitHub settings
